= Dependency Graph

The broker builds a dependency graph from verification results, showing provider/consumer relationships across applications.

== Endpoints

`GET /api/v1/graph`::
Returns the full dependency graph with nodes (applications) and edges (verification relationships).
Optionally filter by environment with `?environment=production`.

`GET /api/v1/graph/applications/{name}`::
Returns provider and consumer dependencies for a specific application.

== Response Model

[source,json]
----
{
  "nodes": [
    { "applicationId": "...", "applicationName": "order-service", "owner": "team-a" }
  ],
  "edges": [
    {
      "providerName": "order-service",
      "providerVersion": "1.2.0",
      "consumerName": "checkout-ui",
      "consumerVersion": "3.0.1",
      "status": "SUCCESS",
      "verifiedAt": "2025-01-15T10:30:00Z"
    }
  ]
}
----
