= Environment Tracking

Track which application versions are deployed to which environments.

== Overview

Teams record deployments to keep the broker informed about the current state of each environment.
This information is used by the Can I Deploy safety check to determine compatibility.

== API

* `POST /api/v1/environments/{env}/deployments` — Record a deployment
* `GET /api/v1/environments/{env}/deployments` — List deployments in an environment
* `GET /api/v1/environments/{env}/deployments/{appName}` — Get deployment for a specific application

See specification: `docs/specs/004-environment-tracking.md`
